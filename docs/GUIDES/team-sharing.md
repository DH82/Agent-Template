# Team Sharing Guide

When expanding this personal template into a shared team template:

1. reduce personal preference in `AGENTS.md` and `CLAUDE.md`
2. adapt the existing `.github/` issue and PR templates to team workflow
3. add shared validation commands to `Taskfile.yml`
4. run structure checks, lint, and tests in CI
5. keep only high-value reusable subagents in `.claude/agents/`

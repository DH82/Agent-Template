# Work Item Template

This template defines the default structure for one generic work item.

- `TASK.md`: goals and done criteria
- `CONTEXT.md`: background and constraints
- `PLAN.md`: execution plan
- `TODO.md`: checklist
- `LOG.md`: work log
- `DECISIONS.md`: decisions and rationale
- `OUTPUT/`: final outputs
- `REFERENCES/`: references
- `SCRATCH/`: disposable material
